# Intellitoggle Website

A responsive marketing website built using Nuxt 4 and Vue 3, based on the Intellitoggle Figma design.

## Tech Stack
- Nuxt 4
- Vue 3
- Vite
- Tailwind CSS

## Features
- Fully responsive (Desktop, Tablet, Mobile)
- Component-based architecture
- Clean and scalable structure
- Pixel-accurate implementation from Figma

## Setup
npm install
npm run dev
